密码重置说明

学生密码重置需要通过 Supabase Dashboard 操作：

1. 登录 Supabase Dashboard

2. 进入 Authentication > Users

3. 找到目标用户，点击 ... 菜单

4. 选择 Send password reset email 或手动修改密码

安全提示
